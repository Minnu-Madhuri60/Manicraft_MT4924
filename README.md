This project is part of the SQL Data Analysis Internship – Task 1.
The aim is to create a Student Management Database using SQL and perform queries to analyze student performance.

🔹 Technologies Used
SQL
VSCode
MySQL Workbench
🔹 Features
Created StudentManagement database
Created Students table
Inserted student records
Performed SQL queries:
Display all students
Average subject scores
Top performer
Students count by grade
Average score by gender
Students with Math score > 80
Update student grade
🔹 Learning Outcome

Learned database creation, table management, data insertion, and analytical SQL queries.
